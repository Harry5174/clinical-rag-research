# Reviewer Concern Mapping

## Status vocabulary

`RESOLVABLE_BY_PAPER_CORRECTION` means a future manuscript change can accurately align a claim with established evidence. `RESOLVABLE_BY_LIMITATION_FRAMING` means the current evidence supports a transparent scope statement, not a new positive finding. `RESOLVABLE_BY_EXISTING_REPRODUCTION_EVIDENCE` is limited to the exact reproduced local protocols. `REQUIRES_NEW_ANALYSIS_OR_EXPERIMENT` must not be answered by an unsupported manuscript claim. `REQUIRES_SUPERVISOR_CLARIFICATION` identifies an ambiguous target or decision. `DEFER_TO_REBUTTAL_DRAFT` is intentionally not drafted in this sprint.

## Cross-cutting response rules

1. Existing reproduction supports only the exact protocol-bound hard-negative and target-subject-filtered aggregates; it does not resolve router, baseline, sensitivity, or deployment concerns.
2. A limitation statement is appropriate where evidence is absent, but it must not substitute for a claimed analysis.
3. Future paper edits should avoid final rebuttal phrasing; this table is an internal mapping and revision-planning artifact.
4. P38 remains deliberately unexpanded until its intended manuscript target is supplied.
